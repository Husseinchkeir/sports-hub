# Sports Hub React Project

Sports Hub is a frontend ReactJS web application for fans who want a simple place to view sports categories, team information, match schedules and feedback options.

The project continues the same idea from the first HTML/CSS phase, but it is now developed with React components and simple interactive features.

## Pages

- Home
- About
- Sports Menu
- Contact

## Main Features

- Responsive design for desktop and mobile
- Reusable React components
- Favorite sport selector
- Match schedule filter
- Search box for matches
- Contact form with validation
- Clean navigation and footer

## Technologies Used

- ReactJS
- Vite
- HTML
- CSS
- Git

## How To Run The Project

Open the project folder in VS Code, then run:

```bash
npm install
npm run dev
```

After that, open the local link shown in the terminal, usually:

```text
http://localhost:5173/
```

## Build Command

```bash
npm run build
```

## Deployment Options

The project can be deployed on GitHub Pages, Vercel or Netlify after pushing the project to GitHub.
