SPORTS HUB REACT - CLEAN RUN VERSION

This clean version does NOT include package-lock.json, so npm will download packages from the public npm registry.

To run:
1. Open this folder in VS Code.
2. Open Terminal > New Terminal.
3. Make sure you are in this folder and can see package.json when typing: dir
4. Run:
   npm install
5. Then run:
   npm run dev
6. Open the localhost link shown in the terminal.

If npm still shows ETIMEDOUT, the issue is internet/network, not the project.
Try mobile hotspot, then run npm install again.
